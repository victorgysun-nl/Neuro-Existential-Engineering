### `.gitignore`

这个文件的作用是确保你本地的隐私数据（如实验日志、临时文件）不会被意外上传到公共仓库。

Code snippet

```
# --- NE-E Private Data ---
# 保护架构师的私有实验记录和黑匣子数据
private_logs/
experiments/private_*.md
personal_audits/
*.log

# --- Editor / OS Files ---
# 排除系统产生的临时文件
.DS_Store
Thumbs.db
.vscode/
.idea/

# --- Obsidian / Knowledge Base ---
# 如果你使用 Obsidian 协作，建议排除个人插件设置
.obsidian/workspace.json
.obsidian/graph.json

# --- Documentation Build ---
# 如果未来生成静态网页（如 GitBook）
_book/
node_modules/
```
